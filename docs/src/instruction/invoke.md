## invoke

### Operation
Invoke function.

### Format
| invoke |
| :----: |

### Forms
```
invoke =
```

### Operand Stack
```
..., [arg1, [arg2...]], function ->
..., result
```

### Description
Invoke `function` with args (`arg1` .. `argN`).

The `result` is the function return value.

### Notes
