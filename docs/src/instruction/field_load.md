## field_load

### Operation
Load object field.

### Format
| field_load |
| :----: |
| index |

### Forms
```
field_load =
```

### Operand Stack
```
..., object ->
..., field
```

### Description
Push the `field` of the `object` onto the operand stack.
The `index` must be a valid **Constant Field Info Pool** index.
