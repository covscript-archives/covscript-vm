## const8_\<n\>

### Operation
Push byte.

### Format
| const8_\<n\> |
| :----: |

### Forms
```
const8_m1 =
const8_0 =
const8_1 =
const8_2 =
const8_3 =
const8_4 =
const8_5 =
```

### Operand Stack
```
... ->
..., <n>
```

### Description
Push the `int` constant *\<n\>* (-1, 0, 1, 2, 3, 4, 5)
onto the operand stack.
